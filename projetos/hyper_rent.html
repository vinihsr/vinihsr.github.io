Hyper-Rent

Plataforma de aluguel de carros focada na distribuição de dados analíticos via API.

O Hyper-Rent é um sistema de gerenciamento de aluguel de veículos desenhado para atuar como uma fonte de extração de dados (Data Source). Além do portal transacional para clientes, a aplicação conta com um painel administrativo voltado para a governança e distribuição de métricas via integrações seguras.

Arquitetura e Stack Tecnológica

A aplicação segue um modelo modular com proxy reverso para balanceamento e isolamento de rotas:

Hyper-Rent/
├── client/           # Interface (React)
├── server/           # Backend API (Express / Node.js)
├── nginx/            # Proxy Reverso e Gateway (Nginx)
└── package-lock.json

Módulos Principais (Data Engineering)

  • Gestão de API Keys: Motor interno de criação, permissão e revogação de chaves criptografadas para acesso seguro aos endpoints por parceiros externos ou pipelines de extração (ETL/ELT).
  • Delivery de Dados Agregados: Endpoints projetados para fornecer dados pré-processados e sumarizados (ex: taxa de ocupação, carros por ano/modelo), mitigando a carga analítica no banco de dados operacional.
  • Filtro de Compliance (LGPD): Implementação de regras de opt-in integradas aos Termos e Condições (T&C). A API restringe programaticamente o retorno de dados, expondo apenas registros transacionais de clientes que autorizaram o uso para fins analíticos.
