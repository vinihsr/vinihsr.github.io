FATEC SJC // APRENDIZADO POR PROJETOS INTEGRADOS (API)

Mirage Group

Um compilado de soluções reais para empresas do mercado.

Minha experiência prática de desenvolvimento e engenharia de software foi forjada na FATEC através de APIs (Aprendizado por Projetos Integrados). A cada semestre, fomos desafiados a solucionar problemas reais de grandes empresas do mercado de tecnologia, assumindo papéis de desenvolvimento e gestão ágil.

Abaixo, documento a progressão técnica da equipe Mirage Group ao longo de 6 semestres.

1º SEMESTRE (AGO/2022) // FATEC SJC

Gestão de Manutenção de Laboratórios

Desafio: Mapear e facilitar a abertura de chamados técnicos para identificar e consertar computadores debilitados nos laboratórios da instituição.

Solução: Aplicação web interativa com drag and drop para reorganizar visualmente as máquinas das salas e gerenciar o estado de manutenção.

  • Stack: Python (Flask), MySQL, HTML5, CSS e JavaScript.
  • Minha Atribuição: Construção e estilização das telas (foco em responsividade), além de contribuir ativamente na documentação que baseou a lógica do drag-and-drop.
2º SEMESTRE (FEV/2023) // FATEC SJC

Gerenciamento de Turmas e Estatísticas Escolar

Desafio: Desenvolver um sistema desktop para substituir um portal instável, permitindo que os professores gerenciassem turmas, atividades, notas e acompanhassem o desempenho estatístico de forma eficiente.

  • Stack: Java, Java Swing e MySQL.
  • Minha Atribuição: Protótipo e desenvolvimento da interface gráfica utilizando Swing e participação na construção da lógica da base estatística. Liderança ativa para contornar desafios de compilação da equipe.
3º SEMESTRE (AGO/2023) // IONIC HEALTH

Rastreio de Processos Regulatórios Empresariais

Desafio: Criar uma plataforma web unificada para gerenciar, documentar e monitorar evidências e etapas de processos regulatórios burocráticos de uma empresa focada em engenharia clínica.

  • Stack: Node.js, TypeScript, React e MySQL.
  • Minha Atribuição: Criação de telas críticas da aplicação (como login/home) e estruturação do feedback contínuo para guiar a tomada de decisões arquiteturais do projeto.
4º SEMESTRE (FEV/2024) // SIATT

Plataforma Integrada de Agendamentos (Híbrido)

Desafio: Solucionar a desorganização de equipes em crescimento no contexto pós-pandemia, coordenando agendamentos que respeitassem diferentes formatos (presencial, online, híbrido) e disponibilidades.

  • Stack: Node.js, TypeScript, React e MySQL.
  • Minha Atribuição: Idealização do MVP, quebra de tarefas (tasks) e desenvolvimento visual do sistema de cadastro de reuniões.
5º SEMESTRE (AGO/2024) // KERSYS

Monitoramento Climático Georreferenciado

Desafio: Desenvolver uma solução mobile para que produtores rurais monitorassem condições climáticas extremas em suas áreas específicas de cultivo, emitindo notificações de impacto.

  • Stack: React Native, Node.js, Firebase (NoSQL) e MySQL.
  • Minha Atribuição: Scrum Master. Gerenciei os ritos ágeis utilizando o Jira Software. Priorizei o backlog, assegurei prazos e blindei a equipe de bloqueios trabalhando diretamente com o Product Owner.
6º SEMESTRE (FEV/2025) // DOM ROCK

Plataforma RLHF: Retreinamento de LLMs

Desafio: Aplicação web para avaliar respostas de Inteligência Artificial em tempo real. O sistema envia prompts para duas LLMs, recebe o retorno via streaming e coleta o feedback humano (RLHF) para retreinar o modelo de cuidado especializado para pacientes com Alzheimer.

  • Stack: Vue.js, Spring Boot, FastAPI, LangChain, Groq (Qwen/DeepSeek), MongoDB e ChromaDB.
  • Minha Atribuição (Desenvolvedor):
    - Definição arquitetural do MongoDB voltada para a fluidez do RLHF.
    - Implementação de Streaming Response para reduzir a latência das LLMs.
    - Construção de rotas integradas para prover dados aos gráficos de performance e de comparação analítica.
